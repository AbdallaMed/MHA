# MHA
Mental Health Awareness Project

This project is a Mental Health Awareness landing page designed to raise awareness and provide a safe way to manage your mental health. The landing page features practical mental health awareness information for people looking to improve their mental well health being or understand more about mental health.

Features

Responsive Design: Mobile-friendly layout using HTML, CSS, Bootstrap, Flexbox.

Social Media Integration: Social media icons (Instagram, Twitter, Facebook) for easy sharing.

Accessible: Designed with accessibility in mind (ARIA labels for social media icons, optimized alt text for images).

Technologies Used

HTML5: Structure and content of the page.

CSS3: Styling and layout, including responsive design with media queries.

Bootstrap: For some basic styling and responsive grid layout.

Remix Icon: For social media and other icons.

Installation

To run this project locally, follow these steps:

Clone the repository First, clone the repository to your local machine:

git clone

Navigate into the project directory cd mental-health-awareness-landing-page

Open the index.html file

You can now open the index.html file in your preferred web browser to view the landing page.

open index.html # On MacOS

OR start index.html # On Windows

(Optional) Customize the content If you'd like to make changes, you can edit the index.html file or update the CSS styles within the <style> tag in the head section.
